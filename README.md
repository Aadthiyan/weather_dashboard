# Weather Dashboard

This is the Historical Weather Data Dashboard "Overview Page" implementation.

## Tech Stack
- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** CSS Modules (Vanilla CSS)
- **Charts:** Recharts
- **Icons:** Lucide React

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Click [Here](https://weather-dashboard-rosy-eight.vercel.app/) to view the dashboard.

## Project Structure
- `src/app/page.tsx`: Main Overview page.
- `src/components/charts/`: Recharts components and mock data.
- `src/components/ChartCard/`: Reusable card wrapper.
- `src/components/Sidebar.tsx` & `Header.tsx`: Layout components.
